# Expense Tracker

Expense Tracker is a responsive web application built using React.js and Vite that helps users manage their daily expenses efficiently.

## Features

- Set and manage budget
- Add expense transactions
- Track total expenses
- Automatic balance calculation
- Search expenses instantly
- Delete transactions
- Real-time updates
- Responsive user interface

## Technologies Used

- React.js
- Vite
- JavaScript (ES6+)
- HTML5
- CSS3

## Installation

Clone the repository

```bash
git clone https://github.com/Ankit787762/expense-tracker.git
```

Navigate to project directory

```bash
cd expense-tracker
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```
